## English

This is a basic lottery system developed in **C++** as part of logic and data structures studies.

## About the project

The goal is to simulate how a lottery system works, where 6 (six) numbers are randomly generated between 1 and 60 and a verification is made it cheking if is there repeated numbers between the 6 (six) numbers drawn.

## Techs

* **Language:** C++

* **IDE:** Visual Studio.

* **VCS:** Git.

## How to run

1. Clone the repo:
`git clone https://github.com/talktodeego/Lottery-System.git`

2. Compile the code:
`g++ lottery.cpp -o lottery`

3. Execute the program:
`./lottery`
